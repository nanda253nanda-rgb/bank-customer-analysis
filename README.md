🏦 Bank Report Analysis Dashboard
📊 Project Overview
This project presents an interactive Bank Report Analysis Dashboard developed using Microsoft Power BI. The dashboard provides a comprehensive overview of banking performance, customer information, financial metrics, and key business insights.
The main objective of this project is to transform raw banking data into meaningful visual insights that help understand customer behavior, financial performance, and overall banking operations.
🎯 Objectives
Analyze overall bank performance using key performance indicators (KPIs)
Understand customer demographics and financial behavior
Analyze customer balances, credits, and financial activity
Identify customer segments and their contribution to the bank
Track important financial metrics through interactive dashboards
Support data-driven decision-making using visual analytics
🛠️ Tools & Technologies Used
Microsoft Power BI
Power Query – Data cleaning and transformation
DAX – Creating calculated measures and KPIs
Data Visualization – Charts, cards, slicers, and interactive reports
📁 Dataset
The dataset contains banking and customer-related information, including details such as:
Customer Information
Customer Age
Gender
Geography / Region
Credit Score
Account Balance
Estimated Salary
Number of Products
Credit Card Status
Active Member Status
Customer Churn / Exit Information
📌 Dashboard Features
📈 Executive Overview
Total Customers
Total Balance
Total Credit
Average Credit Score
Customer Churn Rate
Active and Inactive Customers
👥 Customer Analysis
Customer distribution by age group
Customer distribution by gender
Customer analysis by geography
Active vs inactive customers

💰 Financial Analysis
Balance analysis by region
Credit score analysis
